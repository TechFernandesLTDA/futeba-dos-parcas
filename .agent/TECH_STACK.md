# Tech Stack

## Linguagem

- **Kotlin**: Linguagem principal para todo o código Android.

## Frameworks & Bibliotecas Principais

- **Android Jetpack**:
  - **Navigation Component**: Navegação entre Fragments (Single Activity Architecture).
  - **ViewModel & LiveData/Flow**: Gerenciamento de estado.
  - **ViewBinding**: Acesso seguro às Views do layout XML.
- **Hilt**: Injeção de Dependências.
- **Firebase**:
  - Firestore (Banco de dados).
  - Authentication (Login).
- **Coroutines**: Programação assíncrona.
- **Coil**: Carregamento de imagens (substituindo Glide/Picasso).
- **ThreeTenABP**: Manipulação de Datas/Horas (backport do java.time).

## Ferramentas de Build

- **Gradle Kotlin DSL (`build.gradle.kts`)**: Scripts de build.

## UI

- **XML Layouts**: Sistema de View clássico (Material Components).
- **Material Design**: Componentes visuais padrão Google (Cards, Buttons, Inputs).
